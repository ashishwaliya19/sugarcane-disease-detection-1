## 📌 Project Overview

An image-based machine learning system for automated detection, classification, and identification of sugarcane leaf diseases using a CNN model deployed via Streamlit.

---

## 🦠 Diseases Detected

| Class | Description |
|-------|-------------|
| ✅ Healthy | Normal leaf |
| 🔴 Red Rot | Red discoloration inside stalks |
| 🟤 Rust | Orange-brown pustules on leaves |
| 🟡 Yellow | Yellowing due to infection |
| 🟣 Mosaic | Mosaic discoloration patterns |

## ⚙️ Installation

```bash

cd sugarcane-disease-detection-1
git lfs pull
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
